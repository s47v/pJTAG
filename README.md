# pJTAG

This code allows the Pico to send the correct JTAG sequence to transfer and unlock the SPC58 with a given password. It is based on the code from the dirtyJTAG-pico project

| Pin name | GPIO   | Pico Pin Number |
|:---------|:-------| -          |
| TDI      | GPIO16 | 22         |
| TDO      | GPIO17 | 21         |
| TCK      | GPIO18 | 24         |
| TMS      | GPIO19 | 25         |
| RST      | GPIO20 | 26         |
| TRST     | GPIO21 | 27         |
